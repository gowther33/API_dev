# Need for postman
    1. To test api's we can use our browser for smaller things.
    2. But for complex api calls we need a full front end application
    3. We use tools like postman to test our api's

In Python, the ellipsis (...) is a special object called the "ellipsis object" or "ellipsis literal."
It is used to represent a placeholder or to indicate that the code needs to be filled in.
